Google Colab is a powerful tool researchers and data scientists use to write & execute code, analyze data, and collaborate on projects from anywhere with an internet connection.
It's free to use and requires no setup or installation, making it an ideal platform for data analysis tasks.

In this week's learning activity, we will introduce you to the basics of using Google Colab.
We'll show you how to log in, execute code, write text using Markdown, and export your work for submission.Â 

You don't need to have anything downloaded or installed in advance of this activity.
We will cover everything during the video.

By following along with the video, you'll learn how to:

 - Launch Google Colab
 - Create text using Markdown.
 - Execute code cells.
 - Check your answers with the `Otter` grading system.
 - Submit assignments to BBLearn.

Google Colab is widely used by researchers around the world and is an important tool to have in your data analysis toolkit.
After completing this learning activity, you'll have a strong foundation in using Google Colab and be well on your way to using it to tackle real-world data problems.
Please make sure to respond to the quiz questions as they appear in the video.

Note:

When downloading the files, it's important that you don't unzip them.
If your browser unzips them automatically during the download process, you will need to turn this feature off.
The steps to do this vary depending on which browser you are using (Chrome, Safari, Edge, Internet Explorer, Opera, etc.).
To find out how to disable this feature in your browser, try searching online for a phrase like "how to stop [browser name] from unzipping downloaded files."

If you continue to have difficulty with this, you can schedule a meeting through the booking link to get help from an instructor.
